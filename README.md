# 経験履歴の詳細
私の経験を１年毎に記載しています。
- 2020年: 初めてのHTML。SeaMonkeyでHTMLを書いていたのを覚えてます。1行もJavaScriptが書けなかった。
  - おまけ情報: 小２からPCは好きだった。2020年から不登校期間が始まり、その中でPCとの時間が増えた。
- 2021年: Discord Botなるものを知り、JavaScriptに手を出した。コピペが主でした。
- 2022年: 本格的にBotを作る。自己解析を始め、その調子で自作サイトの制作を開始。初めてのサイトはオフライン環境で使える動画再生ソフトでした。
- 2023年: TypeScriptに乗り換えた時代だったはず。たくさんコードを書くだけではなく、無駄を減らすことにも気が向くように。Node.jsに手を出すようになりました。
  - おまけ情報: 不登校脱却。毎日登校を始めた。
- 2024年: Node.jsネイティブになった。サーバーサイドのプログラムをよく書いたり、ブラウザと連携するアプリケーションを開発するようになった。
- 2025年: 静的コンテンツのみで動作するソフト・運用可能なブログを作ったり、AIでAPI作る時間を削ったりするように。1週間で過去の音楽botを参考に新しい音楽botを作ったりもしました。
  - おまけ情報: １年で高校卒業程度認定試験に合格。
- 2026年: 音楽botの安定運用と改善を継続中。IT業界で働くために学校に入学し、さらにSwiftやJavaの思想を経てクラスに対する理解が深まりました。
- 現時点: ITパスポート楽勝(過去問用紙を学校から渡されて解いたら85点)。週４バイトと並行しながら応用情報目指して勉強中。
